05   Memory

Dual Persistent Memory

🧠
ChromaDB — Semantic Memory
VECTOR STORE — LONG TERM
Stores full conversation turns as embeddings. On each new message, finds 5 most semantically similar past conversations and provides them to the agent as context.

KEY-VALUE — PERMANENT

07   Setup

How to Run

01
Create .env file
Copy .env.example → rename to .env. Fill in your Groq API key.
GROQ_API_KEY=your_key_here
GROQ_MODEL_PRIMARY=llama-3.1-8b-instant
GROQ_MODEL_FALLBACK1=llama-3.3-70b-versatile
GROQ_MODEL_FALLBACK2=llama-4-scout-17b-e3-instruct
02
Install dependencies
pip install -r requirements.txt
03
Terminal 1 — Serve files
Run from project root (not inside test_website/) so test_page.html is reachable.
python -m http.server 3000
04
Terminal 2 — Start API server
python -m api.main
05
Register client
python test_api.py
Copy the CLIENT_ID and API_KEY from output.
06
Paste embed script in your HTML
<script src="http://localhost:8000/widget/chatbot.js"
  data-client-id="YOUR_CLIENT_ID"
  data-api-key="YOUR_API_KEY" defer></script>
07
Open test page
http://localhost:3000/test_page.html
08
If bot doesn't know products — reindex
python check_rag.py --reindex

09   Troubleshooting

Common Issues

ModuleNotFoundError: No module named 'ai'
Create ai\__init__.py as an empty file in the project root. The ai\ folder must have it to be recognized as a Python package.
Bot says "Something went wrong"
Check API server terminal for error logs. Most common: wrong CLIENT_ID or API_KEY in test_page.html, API server not running, or Groq API key expired. Re-run python test_api.py to get fresh credentials.
Product card links show 404
Start HTTP server from project root, not from inside test_website/. The URL builder prepends the client's website_url to relative paths automatically.
Bot doesn't know about products
RAG index is empty or stale. Run: python check_rag.py --reindex
Model chain not switching / empty responses
Check localhost:8000/status. If all 3 Groq models exhausted and Ollama not running, responses will be empty. Wait for midnight UTC reset or start Ollama with ollama serve.
